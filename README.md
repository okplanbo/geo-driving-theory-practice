# Sakartvelo / Georgia Driver's License theory exam practice

Handy app for practice B1 driving test questions before the exam

## Basic commands:

```
pnpm run dev
pnpm run lint
pnpm run build
```

## Built using:

- React
- Vite
- Tailwind
- PostCSS
- Autoprefixer
- Prettier
- Chakra UI
- GitHub Pages
- Semantic release
- pnpm
